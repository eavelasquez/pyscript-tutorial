PyScript Tutorial

The PyScript tutorial is a series of short examples using PyScript.

print('Hello, World! From PyScript')
print('Let\'s try some math! Compute π:') def compute_pi(n): pi = 2 for i in range(1, n): pi *= 4 * i ** 2 / (4 * i ** 2 - 1) return pi pi = compute_pi(100000) s = f'π is approximately {pi:.3f}' print(s)
